# Django REST Framework

## API

> Application Programming Interface의 약자로 프로그램 간 정보 전달을 위한 인터페이스

### API Server

> 기존의 서버는 HTML을 보내주는 것과 달리 정보를 담은 JSON을 보내준다

### REST API

> 기본적인 URI 및 데이터 전송 방식의 규약

| 구분      | 설명          |
| --------- | ------------- |
| GET       | 데이터의 조회 |
| POST      | 데이터의 입력 |
| PUT/PATCH | 데이터의 수정 |
| DELETE    | 데이터의 삭제 |
